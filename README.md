# Price Scraper (Async Python)

## Features
- Scrapes book data (title, price, rating, stock)
- Built using aiohttp + BeautifulSoup
- Async for high performance

## How to Run

pip install -r requirements.txt

python main.py

## Output
- Saves scraped data into database / file

## Use Cases
- Price tracking
- Competitor analysis
- E-commerce monitoring

This scraper can be customized for:
- Amazon price tracking
- Flipkart monitoring
- Shopify stores
